# Statin 不耐受判定標準

> 依據：中華民國血脂及動脈硬化學會 2019 年共識  
> 參考文獻：Chien S-C et al. *JFMA* 2019. doi:10.1016/j.jfma.2018.11.017

---

## 一、健保認定標準

### 不耐受定義（路徑 C 適用）

需**同時**滿足以下條件：

1. ✅ 至少 **2 種不同 statin** 產生副作用
2. ✅ 其中 **1 種**在**最低有效劑量**下仍不耐受
3. ✅ 附病歷紀錄佐證

### 例外：嚴重橫紋肌溶解症

- 只需 **1 種 statin** 即可認定不耐受
- 需附詳細佐證資料

## 三、Myalgia Score（肌肉疼痛評分）

健保規定：路徑 C 的肌肉或肝臟相關副作用，**Myalgia Score > 8 分**方可認定。

### 評分系統

#### A. 症狀部位（0-3 分）

| 分數 | 描述 |
|------|------|
| 3 | 對稱性，累及大腿/髖部 |
| 2 | 對稱性，其他部位 |
| 1 | 非對稱性 |
| 0 | 無肌肉症狀 |

#### B. 時間關係（0-3 分）

| 分數 | 描述 |
|------|------|
| 3 | 開始 statin 後 <4 週出現 |
| 2 | 4-12 週出現 |
| 1 | >12 週出現 |
| 0 | 無時間關係 |

#### C. 停藥反應（Dechallenge）（0-2 分）

| 分數 | 描述 |
|------|------|
| 2 | 停藥後 <2 週改善 |
| 1 | 2-4 週改善 |
| 0 | >4 週或未改善 |

#### D. 再投藥反應（Rechallenge）（0-3 分）

| 分數 | 描述 |
|------|------|
| 3 | 同藥再發 |
| 1 | 不同 statin 再發 |
| 0 | 未再投藥或未再發 |

### 總分判讀

| 分數 | 判讀 |
|------|------|
| **≥9** | Probable statin-related（很可能相關） |
| **7-8** | Possible（可能相關） |
| **<7** | Unlikely（不太可能） |

> ⚠️ 健保要求 **> 8 分**，即 ≥9 分 = Probable

## 四、Statin 不耐受臨床路徑

```
患者主訴 statin 副作用
        │
        ├── 嚴重橫紋肌溶解？
        │   ├── 是 → 停 statin → ezetimibe ≥3 個月 → 申請 PCSK9
        │   └── 否 ↓
        │
        ├── 評估 Myalgia Score
        │   ├── ≤8 分 → 考慮降劑量或換 statin
        │   └── >8 分 ↓
        │
        ├── 換第二種 statin（最低有效劑量）
        │   ├── 可耐受 → 繼續治療
        │   └── 仍不耐受 ↓
        │
        └── 認定 statin 不耐受
            → 停 statin
            → ezetimibe 10mg ≥3 個月
            → LDL-C 仍 ≥100 mg/dL
            → 申請 PCSK9 抑制劑（路徑 C）
```

---

## 五、佐證資料準備

申請路徑 C 時需準備：

- [ ] 第一種 statin：藥名、劑量、健保代碼、副作用描述、發生時間
- [ ] 第二種 statin：藥名、**最低有效劑量**、健保代碼、副作用描述
- [ ] Myalgia Score 評估結果（> 8 分）
- [ ] CK（肌酸激酶）檢驗報告（若有上升）
- [ ] 肝功能（AST/ALT）報告（若為肝臟副作用）
- [ ] 停藥後改善的病歷紀錄
