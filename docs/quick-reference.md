# PCSK9 抑制劑事前審查 — 門診速查卡

> 114/9/1 修訂版 ｜ 列印後放桌上用

---

## ⚡ 30 秒判斷：這個病人能不能申請？

```
□ 有 ASCVD 事件？（MI / 缺血性中風 / 血管再通術）
  └─ 沒有 → ❌ 停，不符合（除非 HoFH）

□ 發病 1 年內開始 statin？
  └─ 超過 → ❌ 不符合

□ Statin ≥3 個月 → 加 ezetimibe ≥3 個月？
  └─ 時間不夠 → ⏳ 再等

□ LDL-C 仍 ≥100 mg/dL？（兩次，間隔 ≥4 週）
  └─ <100 → ❌ 不需要 PCSK9

全部 ✅ → 可以申請！
```

---

## 📊 LDL-C 門檻

| 申請類型 | LDL-C 門檻 |
|---------|-----------|
| **首次申請** | ≥100 mg/dL（兩次，間隔 ≥4 週，3 個月內） |
| **續用** | 下降 ≥30%（vs 申請前基準值） |

---

## 💊 高強度 Statin

| 藥品 | 劑量 |
|------|------|
| Rosuvastatin (Crestor) | **20 mg** |
| Atorvastatin (Lipitor) | **40 mg 以上** |

> 未達此劑量 = 需說明為何是「最大耐受劑量」

---

## 🚫 Statin 不耐受 — 最低有效劑量

| Statin | 最低劑量 |
|--------|---------|
| Rosuvastatin | 5 mg |
| Atorvastatin | 10 mg |
| Pitavastatin | 1 mg |
| Simvastatin | 5 mg |
| Pravastatin | 10 mg |
| Lovastatin | 20 mg |
| Fluvastatin | 20 mg |

> 不耐受需 **2 種 statin**，其中 1 種在最低劑量仍不耐受  
> **Myalgia Score > 8 分**（2019 台灣共識）

## 📁 準備文件清單

### 首次申請
- [ ] ASCVD 事件佐證（出院摘要/手術紀錄）
- [ ] Statin 處方紀錄（藥名、劑量、起迄日期）
- [ ] Ezetimibe 處方紀錄（起迄日期）
- [ ] LDL-C 報告 ×2（≥4 週間隔，3 個月內，均 ≥100）
- [ ] （不耐受）Myalgia Score 評估 + 2 種 statin 副作用病歷

### 續用申請
- [ ] 最近 LDL-C（需下降 ≥30%）

---

## ⏰ 時間計算速算

| 事件 | 起算 | 最快可申請 |
|------|------|----------|
| ASCVD 發病 | Day 0 | — |
| 開始 statin | 發病後儘速 | — |
| Statin 滿 3 個月 | +90 天 | 加 ezetimibe |
| Ezetimibe 滿 3 個月 | +90 天 | 開始抽血 |
| 第一次抽血 | — | LDL-C ≥100 |
| 第二次抽血 | 第一次 +28 天 | LDL-C ≥100 |
| **送件** | 第二次抽血後 | **最快發病後 ~7 個月** |
