# PCSK9 抑制劑健保事前審查申請 SOP

> 適用版本：114/9/1（2025 年 9 月 1 日）修訂  
> 表單：附表二-D「使用健保給付 PCSK9 血脂調節劑事前審查申請表」

---

## 一、申請前檢核（Gate Check）

在開始填表前，先確認病人符合以下**所有**必要條件：

### ✅ 必要條件 1：ASCVD 事件（三選一）

| 事件 | ICD-10 參考 |
|------|------------|
| 心肌梗塞（MI） | I21.x |
| 動脈硬化相關缺血性腦中風 | I63.x |
| 冠狀動脈或其他動脈血管再通術 | — |

> ⚠️ **發病後 1 年內**必須已開始使用最大耐受劑量 statin

### ✅ 必要條件 2：藥物治療路徑（三選一）

#### 路徑 A — 標準路徑（最常見）

```
高強度 statin ≥3 個月
    ↓ 確認 LDL-C 仍高
+ ezetimibe 10mg ≥3 個月
    ↓ 確認 LDL-C 仍 ≥100 mg/dL
→ 可申請
```

**高強度 statin 定義**：
- Rosuvastatin 20mg
- Atorvastatin 40mg（含）以上

> 💡 如果無法達到上述劑量，需在表單詳述「最大耐受劑量」及原因

#### 路徑 B — Statin 禁忌

- 藥物過敏（需提供成分名、藥品名、健保代碼、過敏反應佐證）
- 活動性肝病變（需附佐證資料）
- 持續使用 ezetimibe ≥3 個月，LDL-C 仍 ≥100 mg/dL

#### 路徑 C — Statin 不耐受

需同時滿足：
1. **至少 2 種 statin** 產生副作用（附病歷紀錄）
2. 其中一種在**最低有效劑量**下仍不耐受
3. 副作用類型：
   - 嚴重橫紋肌溶解症（只需 1 種 statin 即可認定）
   - 肌肉或肝臟相關副作用（**Myalgia Score > 8 分**）
   - 其他（需說明）
4. 持續使用 ezetimibe ≥3 個月，LDL-C 仍 ≥100 mg/dL

**最低有效劑量參考**（2019 台灣共識）：

| Statin | 最低有效劑量 |
|--------|------------|
| Rosuvastatin | 5 mg |
| Atorvastatin | 10 mg |
| Pravastatin | 10 mg |
| Lovastatin | 20 mg |
| Fluvastatin | 20 mg |
| Pitavastatin | 1 mg |
| Simvastatin | 5 mg |

## 二、需要準備的資料

### 首次申請

- [ ] ASCVD 事件病歷摘要（含發病日期）
- [ ] Statin 處方紀錄（藥品名、劑量、起迄日期）
- [ ] Ezetimibe 處方紀錄（起迄日期）
- [ ] **最近 3 個月內** LDL-C 報告 × 2 次（間隔 ≥4 週）
- [ ] 兩次 LDL-C 均 ≥100 mg/dL
- [ ] （路徑 B）過敏佐證或肝病資料
- [ ] （路徑 C）Myalgia Score 評估表、2 種 statin 副作用病歷

### 續用申請（每 12 個月）

- [ ] 使用後 LDL-C 下降 ≥30%（與申請前基準值比較）
- [ ] 最近一次 LDL-C 報告

## 三、填表步驟

### 第一區：申請者資料
- 醫院、代碼、日期
- 申請醫師（含醫師證書字號）
- 區分「本次申請日期」vs「首次申請日期」

### 第二區：病人資料
- 基本資料、身分證、保險身分

### 第三區：申請原因（核心）

#### 3.1 ASCVD 事件
- 勾選事件類型
- 填寫**發病日期**（關鍵！須在申請前 1 年內開始 statin）

#### 3.2 藥物路徑
- 勾選路徑 (1)(2)(3) 之一
- 路徑 (1)：填寫 statin 名稱、劑量、治療期間 + ezetimibe 治療期間
- 路徑 (2)：附過敏/肝病佐證 + ezetimibe 治療期間
- 路徑 (3)：填 Myalgia Score、2 種 statin 資訊 + ezetimibe 治療期間

#### 3.3 首次申請 LDL-C（2 筆）
| 項目 | 要求 |
|------|------|
| 報告日期 | 最近 3 個月內 |
| 間隔 | ≥4 週 |
| 數值 | 均 ≥100 mg/dL |

#### 3.4 續用申請 LDL-C
- 與基準值比較下降 ≥30%

### 第四區：藥品選擇

| 藥品 | 成分 | 劑量 | 頻率 |
|------|------|------|------|
| Praluent (Alirocumab) | 75mg/mL | 75mg | Q2W |
| Repatha (Evolocumab) | 140mg/mL | 140mg | Q2W |

## 四、常見退件原因與對策

| # | 退件原因 | 對策 |
|---|---------|------|
| 1 | LDL-C 報告不足 2 次或間隔 <4 週 | 確保兩次報告間隔 ≥28 天 |
| 2 | Statin 使用未滿 3 個月 | 計算精確日期，不足就等 |
| 3 | Ezetimibe 使用未滿 3 個月 | 同上 |
| 4 | 未附 ASCVD 事件佐證 | 附出院摘要或手術紀錄 |
| 5 | 發病超過 1 年才開始 statin | 確認時間軸，必要時說明原因 |
| 6 | 不耐受路徑未附 Myalgia Score | 務必完成評分並附佐證 |
| 7 | LDL-C <100 mg/dL | 不符合門檻，無法申請 |
| 8 | 續用 LDL-C 降幅 <30% | 考慮調整劑量或合併治療 |

## 五、時間軸範例

### 範例：標準路徑首次申請

```
2025/01/15  MI 住院（ASCVD 事件）
2025/01/20  開始 Rosuvastatin 20mg（住院中）
    ↓ ≥3 個月
2025/04/25  確認 LDL-C 仍高，加 ezetimibe 10mg
    ↓ ≥3 個月
2025/07/28  抽血①  LDL-C = 118 mg/dL（≥100 ✅）
    ↓ ≥4 週
2025/08/28  抽血②  LDL-C = 112 mg/dL（≥100 ✅）
2025/09/01  → 送出事前審查申請
```

### 範例：不耐受路徑

```
2025/03/01  缺血性中風住院
2025/03/05  開始 Atorvastatin 10mg → 2 週後肌肉痛
2025/03/20  換 Rosuvastatin 5mg（最低劑量）→ 仍有肌肉痛
            Myalgia Score > 8 分
2025/04/01  停 statin，開始 ezetimibe 10mg
    ↓ ≥3 個月
2025/07/05  抽血①  LDL-C = 145 mg/dL
2025/08/05  抽血②  LDL-C = 138 mg/dL
2025/08/10  → 送出事前審查（路徑 C）
```
